Visualizing Arrival Predictions

Introduction

Real-time transit arrival predictions can be a great tool for riders. Or they can make transit less friendly with clunky and static interfaces. We created a visualization for transit users to get next bus arrival information at any stop along a route at a glance. The following interactive map shows our visualization.

  • Zoom in and out
  • Drag the map
  • Click on a stop

The map uses data collected on the Bear Transit campus shuttle system at UC Berkeley from 8am to 10am on July 11th, 2013. The time has been speed up by a factor of 5.

How to use it

Compare the color surrounding a stop with the legend to infer the time until the next bus arrival. Click on a stop to pop-up a window with numeric information. This particular window just contains the stop name and the time until the next arrival in minutes. Other useful information that one could include is: distance between the stop and the next bus to arrive, ammenities surrounding the stop, etc.

In addition, when a stop is clicked, a gray circular catchment area appears. If you are located within the circle, you should be able to catch the next bus at that stop by walking at a casual pace.

How it works

In a real setting, the application combines real-time bus positions and arrival prediction information to forecast each bus trajectory. A piece-wise linear function constructed from these trajectories defines the color mapping along the route. The next figure provides a graphical depiction of how this process would work over time.
